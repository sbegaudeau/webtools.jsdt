Javascript Development Toolkit (JSDT)

Adding a library to the Global Scope

  To add a JavaScript Library to a project follow these instructions:

1. Right-click on the project and select Properties
 

2. Expand the JavaScript properties page in the left navigation bar and select the JavaScript Libraries page. Then click the Add Runtime Library to bring up the Libraries Selection dialog.
 

3. In the Add Library dialog select an available library and click Next.
 

4. Some libraries may have additional configuration pages. Fill them out as needed. Complete the library specific configuration by clicking Finish on the last page.
 

5. Finally, click OK to close the JavaScript Libraries page.
