Rhino Tab Group

The Rhino tab group is the complete group of tabs for configuring a Rhino JavaScript launch configuration. The group consists of the following tabs:

  • Main tab - this tab allows you to specify the script and optional project to launch, as well as set interpreter options
  • Include tab - this tab allows you to specify additional scripts / folders of scripts to include in the launch
  • Source tab - this tab allows you to configure where script source is located
  • Common tab - this tab allows you to configure common settings for the launch configuration
