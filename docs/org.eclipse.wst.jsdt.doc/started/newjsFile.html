Adding a JavaScript file to a Project

  To add a JavaScript source file to a project follow these instructions:

1. Right-click on the project and select New --> Other
 

2. In the New Wizzard navigate to the Web Folder and exapnd it to reveal the JavaScript entry. Select JavaScript and click Next.
 

3. Type a file name name in the File name field and click Finish.
