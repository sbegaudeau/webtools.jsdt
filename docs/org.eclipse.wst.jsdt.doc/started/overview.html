JavaScript Development Toolkit (JSDT)

Overview

 

The JSDT provides JavaScript support in standalone JavaScript files (.js) as well as support for JavaScript embedded in HTML or JSP.

To edit only JavaScript source files with NO HTML support, create a new JavaScript project from the New Project Wizard. For additional information on stand alone JavaScript projects see Pure JavaScript..

JavaScript support in HTML or JSP does not require any additional configuration. Just create a Static Web or Dynamic Web Project, create your HTML or JSP pages, and begin editing. For additional information on JavaScript embedded in JSP or HTML.
